GWAS on UKB RAP

Author

Anastazie Sedlakova

Published

November 3, 2021

On this page

  • Learning Objectives
  • Video

Webinar about how to run Regenie (GWAS executable) on the Research Analysis Platform.

Learning Objectives

  1. Describe how regenie utilizes genotype and phenotype information for GWAS

  2. Prepare input files of merged assay genotypes

  3. Prepare phenotype information for use in regenie

  4. Perform single variant association tests on human genomic data using regenie

  5. Explain next steps for GWAS analysis on the RAP platform
